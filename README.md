# COLABRIQ Decentralized Knowledge Graph

## Introduction

This is the open source reference edition of COLABRIQ's Decentralized Knowledge Graph and information sharing protocol. This component is utilities, some facilities for loading TURTLE files and other classes such as a reference RDF client. Please agree to LICENSE.md and read CONTRIBUTING.md before using this code.
